# todo-list
Web Application with Spring Boot + Angular JS for short course in UFCG

## Getting started

1. Install [Node JS] (https://nodejs.org)

2. Install [Bower] (http://bower.io/), just run:
    ```console
    npm install -g bower
    ```

3. Go to the webapp folder and run:
    ```console
    bower install
    ```

4. Have fun!
